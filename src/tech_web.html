Công nghệ Web: HTML & CSS

HTML (HyperText Markup Language)

HTML là ngôn ngữ đánh dấu được sử dụng để tạo cấu trúc của một trang web. Nó giúp xác định các thành phần như tiêu đề, đoạn văn, hình ảnh, liên kết, bảng biểu, biểu mẫu và nhiều thành phần khác.

CSS (Cascading Style Sheets)

CSS là ngôn ngữ dùng để định dạng giao diện của trang web. Nó giúp kiểm soát màu sắc, kiểu chữ, bố cục và hiệu ứng của các phần tử HTML, từ đó làm cho trang web trở nên đẹp mắt và thân thiện hơn với người dùng.
